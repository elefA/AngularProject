 This project was built for an interview question with a company.

It is about CRUD screens for employees and attributes. 

Then the user could navigate to a new screen where a google map was integrated that showed the employees based on their address

---------------------------------------------------------------------------------------------------------

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 12.0.4.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.
